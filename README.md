# ChickenGame
使用 C++ 開發一款 2D 射擊遊戲，運用物件導向設計，模組化物件並提高遊戲的可維護性

### 遊戲Demo
[Demo影片](https://youtu.be/O9V5JIe25Y4)

### 遊戲說明
使用WASD操控角色(烤雞)，角色會不斷發射子彈(雞腿)，玩家需要利用子彈擊破迎面而來的敵人(叉子)來獲得金幣。
使用金幣可購買右側選單的道具，道具包含以下四種:
1. 防護罩:保護角色不會受到敵人攻擊一次
2. 魔法藥水:使角色的血量回到前一個狀態
3. 子彈*3:子彈增加為三倍
4. 子彈巨大化:子彈變得較大

### 遊戲設計
* 角色(烤雞)的顏色會隨著血量減少而改變，最後變為黑色
* 敵人(叉子)為隨機生成，並且會隨著遊玩時間移動得更快，生成的更快
* 不論是成功擊潰敵人、被敵人攻擊成功或是購買道具，都有不同的音效
* 遊戲背景以及角色，都是自己設計
